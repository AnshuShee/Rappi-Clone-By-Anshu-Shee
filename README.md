# Rappi Clone By Anshu Shee

This is a front-end UI clone of the Rappi website, built using HTML and CSS.

## Features
- **Landing Page (`index.html`)**: The main interface displaying restaurants and services.
- **Login Page (`login.html`)**: The authentication interface.
- Responsive design considerations using standard CSS.

## Technologies Used
- HTML5
- CSS3

## How to Run
1. Clone this repository to your local machine.
2. Open `index.html` or `login.html` in your web browser to view the pages.
