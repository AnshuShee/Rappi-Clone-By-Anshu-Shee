# Rappi Clone

This project is a detailed front-end Web UI clone of the popular Latin American mega-app, **Rappi**. Built primarily with standard HTML5 and CSS3, this clone focuses on recreating the visual aesthetic, layout, and user experience of Rappi's core web application interface.

## About Rappi
Rappi is a consumer tech company that provides a wide range of logistics and delivery services. From restaurant food delivery and grocery shopping to pharmacy runs and e-commerce, it acts as an "everything app." This clone aims to capture the vibrant, user-friendly interface that makes Rappi so successful in engaging its users.

## Features Included
- **Landing Page (`index.html`)**: Features the layout for discovering restaurants, markets, and various services provided by Rappi. It includes navigation menus, promotional banners, and category showcases.
- **Alternative Landing Page (`index1.html`)**: An additional view or variant of the home page highlighting different UI components.
- **Authentication Interface (`login.html` & `login.css`)**: A complete UI mockup of the login and sign-up process, retaining Rappi's branding and styling.
- **Custom Styling (`style.css`)**: Contains all the custom CSS used to structure the layouts, typography, specific color palettes, and responsive design behaviors.
- **Assets (`restaurants.png`, `super.png`)**: Custom graphics and placeholders used to mimic the real application's imagery.

## Technologies Used
- **HTML5**: For precise and semantic structuring of web pages and components.
- **CSS3**: For all stylistic elements, including flexbox layouts, hover animations, color schemes, and responsive design rules.

## How to Run the Project
Since this is a static front-end project, no build tools or servers are required to view the primary functionality:
1. Clone this repository to your local machine using `git clone`.
2. Navigate to the project directory.
3. Simply double-click `index.html` or `login.html` to open it directly in your default web browser.
